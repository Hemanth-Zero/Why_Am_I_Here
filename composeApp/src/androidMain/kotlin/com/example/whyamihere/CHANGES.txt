• Brand-new foreground Service that runs the break countdown.
• Accepts EXTRA_DURATION_SECS, EXTRA_PACKAGE_NAME, EXTRA_APP_NAME.
• Ticks every second, posting a "Break time ⏳ – Overlay returns in M:SS"
  notification via postBreakTimerNotification().
• When countdown reaches 0 it re-calls OverlayService.show() to
  retrigger the overlay for the same package.
• Exposes companion val isRunning (Volatile) so AppMonitorService can
  check whether a timer is in progress and skip overlay logic.
• onDestroy() cancels the notification, so if the app/service is killed
  the countdown notification disappears automatically.
